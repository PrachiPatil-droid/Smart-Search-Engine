# Smart-Search-Engine
Smart Search Engine implemented using Trie in C++. Supports prefix search, auto-complete, frequency-based ranking, and deletion. Includes a minimal HTML + JavaScript demo to visualize Trie behavior.


## Features

- Insert a word into the Trie
- Search for words with a given prefix
- Auto-complete suggestions ranked by frequency
- Delete a word from the Trie
- Minimal HTML + JS demo for live visualization

## Tech Stack

- C++ → Core Trie implementation
- HTML + JavaScript → Optional demo to visualize auto-complete and prefix search


## Project Structure
main (branch)
├── c++_code/
│   └── code.cpp
├── project/
│   └── index.html
└── README.md


## How to Run

1. Open `project/index.html` in any browser
2. Type a word in the search box to see suggestions
3. Click "Insert Word" to add a word to the Trie
4. Click "Delete Word" to remove a word from the Trie
